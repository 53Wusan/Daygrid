# 日格 DayGrid

A 15-minute grid-based time tracking app.
一款基于每日15分钟时间记录工具

## Features

- 96 slots per day (15-minute precision)
- 2-level mandatory tags (Category / Event)
- Daily & Weekly statistics
- Category → Subcategory analysis
- Dark mode
- Local persistence

## Philosophy

Understand your life in 15-minute units.
